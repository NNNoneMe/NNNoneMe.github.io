发表于 2018-06-20

写在前面的话

部署GitHub（码云）

  1. 拥有一个GitHub（码云）账号

    注册一个GitHub（码云）账号，你有也行，登录进去。（建议英文不好的用码云）

  2. 新建一个项目（以GitHub为例）

    选择New repository ——>在Repository name下输入项目名：用户名.github.io（必须这样写）——>其他默认不用改，勾选Initialize this repository with a README——>选择Create repository

  3. 打开page服务

总结

终于从零到有的搭建了自己的博客，我最后选择用的码云，GitHub上也有，但是没有更新了。因为码云是国内的，使用GitHub总感觉有点慢（你懂的）。此文如果从头看到尾的话，对我来说是一个提醒，当然如果能帮到一些人我也很开心。博客的评论功能还没有实现，我也在慢慢摸索当中。未来也会一直更新博客，记录自己的学习之路。
